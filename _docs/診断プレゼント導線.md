# 診断プレゼント導線 実装ドキュメント

全15LP（`01-brand`〜`13-nittokomasen`）に埋め込んだ集客導線の説明。関根さんが場所・仕組み・編集箇所を把握するための資料。
※このフォルダ `_docs/` は `_` 始まりのため **GitHub Pages（Jekyll）は公開しません**（サイトには出ない）。

作成：2026-07-19

## 1. 何をする導線か（フロー）

```
各LPを読む
  ├─（固定導線）画面のどこかで「診断」への誘導が出る
  │     ① CTA①セクション内の「◯◯合格可能性診断を受ける」ボタン
  │     ② 中盤スクロールで右下に出るフロートボタン
  │     ③ 診断誘導モーダル（下記トリガーで1回だけ表示）
  ↓
LP最下部の【埋め込み診断】へページ内スクロール
  ↓
3問（偏差値／苦手科目／残り月数）に答える → その場で「合格可能性%＋最優先課題」を表示（登録不要）
  ↓
結果画面に「LINE登録で〈LP固有の教材〉プレゼント」→ LINE登録（＝CV）
  ↓
（別途）LINEで教材配信・体験ページ・無料カウンセリング予約へ
```

- 最優先CTAは従来どおり **LINE登録／カウンセリング予約**。診断はそれを邪魔しない“サブ導線”として設計。

## 2. どこにあるか（各LPの `index.html` 内）

すべて各LPの `index.html` に**自己完結**で入っている（共通CSS/JSは非改変。`shared/` は触っていない）。HTMLコメントの目印で探せる。

| パーツ | 目印（コメント／id／class） | 位置 |
|---|---|---|
| 診断本体＋専用CSS | `<!-- ======= 埋め込み診断（…） ======= -->` → `<style id="shindan-css">` ＋ `<section id="shindan">` | 最終「LINE登録ブロック」の直前 |
| 誘導①（インラインボタン） | `class="shindan-inline-cta"` | CTA①セクション内（`counsel-cta` の直前） |
| フロートボタン | `<!-- 診断フロート … -->` / `id="shindanFloat"` | sticky CTA の直前 |
| 誘導モーダル | `<!-- 診断誘導モーダル … -->` / `id="shindanModal"` | フロートの直前 |
| プレゼント表記 | `<section id="line-cta">` 内の `line-gift-name` 等 | 最終LINE登録ブロック |
| 制御JS | `<script>/* 埋め込み診断 … */ … </script>` | `</body>` 直前 |

## 3. モーダルの表示条件（重要）

`index.html` 末尾JSの `/* 診断誘導モーダル … */` ブロック。

- **1人あたり合計1回だけ**表示（`modalShown` フラグを全トリガーで共有）
- トリガーは**2つのみ**：
  1. **①スクロール**：後半再導入（`.deep-intro`）セクションの下端が画面上端を120px超えて上に抜けたら
  2. **②戻る操作**：`history.pushState`＋`popstate` で、まだ未表示ならブラウザの戻るで表示（`scrollRestoration='manual'` で最上部ジャンプ防止）
- 閉じる：×／「あとで見る」／背景タップ／「診断を受ける（＝閉じて診断へスクロール）」

## 4. LP別に変わる部分（編集する箇所）

同じ構造で、**校名とプレゼント名だけ**LP別。編集するなら各LPの該当文字列を直接変える。

- 校名ラベル：診断タイトル `shindan-title` / 結果 `shindan-result-eyebrow` / モーダル `shindan-modal-title` / 誘導・フロート文言（例「早稲田」「GMARCH」「第一志望」等）
- プレゼント：`shindan-gift-name` と `line-gift-name`（例「早稲田英語 頻出テーマ攻略教材」「塾費用まるわかり 比較資料」等）
- 診断の設問・採点ロジック・見た目CSSは全LP共通。

**一括生成／再適用**：`_docs/apply_shindan.py`。`05-soukei` を雛形に、スクリプト内の `LPS` 辞書（LP→校名・プレゼント名・説明）を編集して実行すると14LPへ再注入できる（既に `id="shindan"` があるLPはスキップ）。
