最終更新：2026-07-21

## 0. 全体の前提

- 導線：**LP内で診断 → 結果表示 → LINE登録でプレゼント → 体験URLへ**
- **プレゼント＝全LP「教材」に統一**（BOOK/図鑑案は不採用）
  - 理由：図鑑は比較検討モードに入らせ、LINE登録率は上がってもカウンセリング受講率が下がるため
- **教材＝ダウンロード物ではなく、URL遷移でのインタラクティブ学習体験**（実際のFAST-UPの学習環境を体感させる）

### 体験でできること（この4つのみ）
1. **毎日の計画**
2. **独自教材（ライブラリ上）**
3. **ミニ模試**（セクションごとの確認テストをWEB上でクイズ形式）
4. **学習管理**（学習時間・ミニ模試の理解度などを振り返れる）

> 注意：**週1固定面談・FAST-UP AI は体験では提供できない**ため、診断結果の接続先に使わない。

## 3. グループB 結果4タイプ（体験4機能と1:1で対応）

**設計原則**：4タイプすべてFAST-UPが提供し、かつ体験で必ず届けられる機能に接続する。他カテゴリの塾（映像授業・集団授業等）に誘導する選択肢は作らない。塾名は一切出さない。

| タイプ | どんな人 | 接続する体験機能 | オファー文（回収） |
|---|---|---|---|
| 毎日管理型 | 今日やることが決まらないと動けない | 毎日の計画 | 毎日やることが決まる状態を、そのまま体験できます |
| 教材特化型 | 何をどの順でやるかが定まらない | 独自教材ライブラリ | 全科目の教材と進める順番を、そのまま開放します |

| データ分析型 | 勉強しているのに伸びている実感がない | 学習管理 | 学習時間と理解度が可視化される画面を体験できます |

## 4. 結果の出力構造（全15LP共通・4ブロック）

画面には①〜④の下書きラベルや文字数表記は出さない。見出しは下記のとおり。

```
① 結果            … 合格可能性%（A） or サポートタイプ（B）＋判定文
② 見出し「あなたの現状」      … なぜその結果なのか（約100字）
③ 見出し「これからやるべきこと」 … 次に何をやるべきか（約100字）
④ オファー         … 教材の価値訴求 → LINE登録
```

### ②③の分岐軸（確定・2軸）
- **グループA**：スコア3段階（低〜54／中55〜74／高75〜）**×** 期間2種 ＝ **6パターン**
- **グループB**：4タイプ **×** 期間2種 ＝ **8パターン**
- **期間の定義**：Q2が「3ヶ月以内／4〜6ヶ月」＝**短期**、「7〜12ヶ月／13ヶ月以上」＝**長期**

### 期間によるロジックの違い（重要）
| | ロジック |
|---|---|
| **短期** | 時間がないので**勉強する範囲を絞る**。「出るところだけを勉強する」 |
| **長期** | まだ直前期の「対策」の段階ではない。**志望校に必要な勉強を早くから始める**ほど合格率が上がる |

> 長期側で「志望校の出題に合わせた対策」「対策を行う」は使わない（残り半年でやることのため不自然）。

### ③アドバイスの文章ひな型（山本さん確定）
【指示】→【入試の事実】→【効果】の3文構成。**行動手順（1日1セクション等）は書かない**。
> 例：やることを絞ってください。私大入試で問われるのは、限られた範囲です。出題される範囲から重点的にやることで、最短での合格が可能になります。

理由：直後のオファーで渡すのは**教材そのもの**であり、進め方の管理ではない。行動手順を書くとオファーと噛み合わない。

### 使用禁止の表現（不自然・AI的なため）
積む／積み上げる／積む順番／射程内／完成度／〜が前提になります／見える化／分散／再現できる／定着不足／管理の仕組み
→ 日常の言葉（勉強する・仕上げる・慣れる・決まっている 等）で書く。

### ④オファーの構成
```
タイトル ： {特化ラベル}の独自教材を、体感してみませんか？
サブタイトル： FAST-UP{選んだ苦手科目}総合　無料プレゼント   ← 科目連動
補足   ： 全科目そろう独自教材の中から、あなたが選んだ〈{科目}〉をお渡しします。
本体   ： 左＝教材モックアップ画像（後ではめ込み）／右＝チェック3つ
         ✓ 私大専門で7年、{対象}の出題傾向を分析し続けて作成
         ✓ 感動する！圧倒的な分かりやすさ
         ✓ セクションごとの確認テストで、定着まで
CTA    ： LINE登録で教材を受け取る
```
> 「体験できる」等の体験訴求は**書かない**。教材そのものの価値だけを訴求する。
> 全科目用意はあるが、**渡すのは選んだ苦手科目1つ**。
