# プレゼント教材 一覧（確定版）

最終更新：2026-07-24 ／ この文書が**教材名・必要数の唯一の正**（`診断設計_確定仕様.md` §5 の旧表より優先）。

## 前提

- 各LPの診断の最後で、利用者は**苦手科目を1つ**選ぶ（**英語 / 国語 / 社会 / 数学** の4択）。
- LINE登録すると、**選んだ科目の教材1冊**を渡す。
- 教材は「ダウンロード物」ではなく、URL遷移でのインタラクティブ学習体験（毎日の計画／独自教材ライブラリ／ミニ模試／学習管理）。
- したがって**1LPにつき4科目分の教材が必要**。ただし大学特化以外の8LPは同じ教材を共有する。

## 教材名の付け方

- **大学・大学群特化 7LP**：`FAST-UP {大学}必出 {科目}`（例：FAST-UP 早慶必出 英語）
- **その他 8LP**：`FAST-UP 私大逆転 {科目}`（例：FAST-UP 私大逆転 英語）※8LP共通

- 数学は私大文系では不要なケースもあるが、診断は4科目すべて選べる仕様のため、**全ファミリー4科目そろえる前提**。数学を外すなら診断の苦手科目選択肢（各LPの `index.html` 内 `SUBJ` と Q4/Q5）から「数学」を除く必要がある。
